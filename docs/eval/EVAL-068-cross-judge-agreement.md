## Summary

Re-scored 3 faculty sweep JSONLs (reflection / perception / neuromod) using
a non-Anthropic judge (Together.ai: meta-llama/Llama-3.3-70B-Instruct-Turbo)

## Disagreement Analysis

### Where judges diverge

**Reflection** (86.0% — only 14 disagreements):  
Disagreements cluster on ambiguous tasks where the agent acknowledges it
cannot use tools vs. attempting anyway. Llama-70B is more lenient on
"honest no-tool acknowledgements" (scores them 1.0); sonnet-4-5 scores
them 0 when the rubric says "must use the list_dir tool". Example:
- `clean-08-code-digest`: sonnet=0.0, llama=0.5 — agent says it can't
  read the file; sonnet penalizes for not calling read_file; llama accepts
  the honest inability statement.

**Perception** (75.0% — 25 disagreements):  
Disagreements on ambiguous/structured tasks where the expected signal is
subtle (risk marker detection, multi-entity extraction). Llama-70B
applies a "good-faith effort" heuristic; sonnet-4-5 follows the rubric
strictly. Example:
- `structured-04-multi-entity`: two cells get opposite scores from the
  two judges — classic sign of rubric ambiguity.

**Neuromod** (71.0% — 29 disagreements):  
Highest disagreement rate. Dynamic tasks (retry loops, budget-aware
behavior) involve multi-step reasoning. Llama-70B assigns partial credit
(0.5) frequently; sonnet-4-5 uses binary thresholding. Example:
- `dynamic-08-budget-aware`: sonnet=0.1-0.2, llama=0.5-0.8 on the same
  agent response — judge diverges on "partial success" interpretation.

## Root Cause Hypothesis

Two consistent patterns explain the 77.3% overall rate:

1. **Rubric literalism gap**: sonnet-4-5 reads the rubric strictly;
   llama-70B applies more contextual / good-faith scoring. Tasks with
   tool-use rubrics (must call X) see systematic sonnet-0 / llama-1
   split when the agent declines politely.

2. **Partial-credit divergence**: llama-70B outputs scores in {0.0, 0.5,
   1.0}; sonnet-4-5 in {0.0, 0.1, 0.2, ..., 1.0}. When binarized at
   0.5, the llama 0.5s become 1 while the sonnet 0.1–0.4s become 0.

## Verdict

Agreement 77.3% < 80% threshold → **EVAL-072 filed** (judge methodology
investigation). The Anthropic results from EVAL-063/064 should be treated
as **preliminary** until EVAL-072 resolves whether the divergence is a
rubric problem, a judge liberality difference, or a systemic Anthropic
bias.
