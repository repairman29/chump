# chump v0.1.2 — 2026-05-15

## What changed

Patch release that ships all commits since v0.1.1 (2026-04-30), most notably:

- **`chump --version`** handler (INFRA-148, merged 2026-05-02) — v0.1.1 binary lacked
  this handler and fell through to the Ollama agent loop, failing every FTUE CI run.
  Root cause: brew tap pinned v0.1.1 (2026-04-30) while the handler landed 2026-05-02.
- Runtime assertion framework (CREDIBLE-065): `assert_json_shape`, `assert_gap_valid`,
  `assert_lease_held` wired into `gap claim` / `gap ship` paths with ambient event emission
- lease_is_fresh sentinel fix (INFRA-1356): age=0s now treated as fresh; -1 for missing heartbeat
- merge_state_status column in github_cache.db (INFRA-1368): kills dominant cache_miss source
- Numerous fleet-coordination, PWA, and observability improvements

## Upgrade

```bash
brew update && brew upgrade chump   # via repairman29/chump tap
chump --version   # should print chump 0.1.2
```

## Structural fix

INFRA-1373 filed: release-staleness CI gate so > 14-day drift between main HEAD
and latest GitHub release gets caught automatically.
