#   <script-path-or-step-name-fragment>  # reason: <one-line why> [gap: <ID>]

#   - If a gate's run: command contains the entry → allowlisted
#   - If a gate's step name contains the entry → allowlisted
#   - Exact match preferred; substrings accepted for compound step names
