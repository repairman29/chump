# scripts/ci/set-e-exemptions.txt — INFRA-1492

# Scripts exempt from the "set -e in first 30 lines" gate.
# Two categories:
#
#   SOURCE — library files meant to be sourced by other scripts.
#            Adding `set -e` at the top of a sourced file propagates
#            it to the calling shell, which may not be desirable.
#            These files still rely on callers having set -e.
#
#   LATE   — scripts that have `set -e` in their file, just after line 30
#            due to a long copyright/comment/shellcheck-disable header.
#            They ARE protected; the placement is cosmetic, not a bug.
