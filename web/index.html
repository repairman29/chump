First-time setup

      
Welcome to Chump

      

      

        
      

      

        

          
Set up local AI in a few steps: we use Ollama by default (private, on your machine). Pick a model size that fits your disk — or point at LM Studio / MLX / any OpenAI-compatible URL in the next screen.

          

            
            
          

        

        

          
Skip setup? Cowork expects a working .env and a reachable inference server. Only skip if you have already configured Chump.

          

            
            
          

        

      

      

        
Install Ollama if you want the default path, then verify it’s available. Keep the Ollama app or ollama serve running while you chat. (If you use another server only, use the button below.)

        

          
          
          
          
        

        

          
        

      

      

        
We’ll create Chump’s folder under your user Application Support (or the equivalent on your OS). Chat sessions and SQLite live there — no git clone required.

        

        
        
        

          Advanced — OpenAI-compatible API base
          
Leave empty for Ollama at http://127.0.0.1:11434/v1. Otherwise set your server URL (must include /v1 if your stack expects it).

          
          
        

        

          
        

        

          
        

      

      

        

          
Download the model with Ollama. This is usually the longest step; progress streams below. You can skip if you already ran ollama pull for this tag.

          

            

          

          

            
          

          

            
            
          

        

        

          
You’re using a custom API base, so there’s no Ollama model to pull here. Make sure that server is running before you start Chump.

          

            
          

        

        

          
        

      

      

        
Start the Chump engine. We wait for a healthy API before closing this screen — you’re almost there.
